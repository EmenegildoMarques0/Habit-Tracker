# Habit Tracker - Sistema de Gerenciamento de Hábitos

Este é um sistema de gerenciamento de hábitos desenvolvido com **Laravel** no back-end, **Vue.js** no front-end e **MySQL** como banco de dados. O objetivo é permitir que os usuários gerenciem seus hábitos, acompanhem seu progresso e recebam lembretes.

## Funcionalidades

- **Cadastro de Usuários**: Registro e autenticação de usuários.
- **Gerenciamento de Hábitos**: Criação, edição e exclusão de hábitos.
- **Acompanhamento de Hábitos**: Marcar hábitos como concluídos, visualizar progresso.
- **Relatórios**: Visualização de estatísticas de hábitos através de gráficos.
- **Notificações**: Sistema de notificações para lembretes.
- **Gamificação**: Recompensas e níveis conforme o progresso do usuário.

## Tecnologias Utilizadas

- **Laravel**: Framework PHP para o back-end.
- **Vue.js**: Framework JavaScript para o front-end.
- **MySQL**: Banco de dados relacional.
- **TailwindCSS**: Framework CSS para estilização.

## Requisitos

- **PHP** >= 8.1
- **Composer**
- **Node.js** >= 16.x
- **NPM** ou **Yarn**
- **MySQL** >= 5.7
